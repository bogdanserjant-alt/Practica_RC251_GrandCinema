ZIUA 3 – DOCUMENTAȚIE

1. Reguli despre formulare HTML:

* Fiecare formular se creează cu tag-ul `<form>`.
* Pentru introducerea datelor se folosesc `<input>`, `<textarea>` sau `<select>`.
* Fiecare input ar trebui să aibă atributul `name` pentru a putea trimite datele.

2. Box Model (Margin vs Padding):

* Margin = spațiul din exteriorul elementului (distanța față de alte elemente).
* Padding = spațiul din interiorul elementului (distanța dintre conținut și margine).

3. Selector de clasă în CSS:

* Se scrie cu punct (`.`) înaintea numelui clasei.
  Exemplu:
  .card {
  background: #111;
  }
